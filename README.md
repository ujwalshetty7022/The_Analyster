✨ The Analyster
AI-Powered Data Platform for Instant Insights

Turn messy spreadsheets into clear decisions — using simple prompts.

🚀 What is The Analyster?

The Analyster is an AI-powered data analysis platform that lets you interact with your data like you chat with a human.

No SQL. No formulas. No complexity.

Just upload your file, ask a question, and get results instantly.

⚡ Why It Matters

Data is everywhere — but insights are not.

Most tools:

Require technical skills
Take time to learn
Slow down decision-making

The Analyster removes that barrier.

→ Anyone can analyze data
→ Anyone can make decisions
→ Instantly

🧠 How It Works
Upload → Ask → Analyze → Visualize
📂 Upload your dataset (Excel / CSV)
💬 Ask questions in plain English
⚙️ AI processes your query
📊 Results appear in clean tables
💡 Example Prompts
"Show total revenue by month"
"Top 5 customers by sales"
"Which product has highest profit?"
"Average growth rate per quarter"

⚙️ Quick Start
1. Clone the repo
git clone https://github.com/your-username/analyster.git
cd analyster
2. Start Backend
cd backend
python -m venv venv
source venv/bin/activate

pip install -r requirements.txt
uvicorn main:app --reload
3. Start Frontend
cd frontend
npm install
npm start
